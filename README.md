# 🎬 Netflix Clone

A fully responsive Netflix clone built using HTML, CSS, and JavaScript.

## 🚀 Features
- Dynamic movie data using TMDB API
- Trending, Top 10, Bollywood, K-Dramas sections
- Interactive banner with search functionality
- Movie modal with details, cast, and similar movies
- Responsive design (mobile + desktop)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- TMDB API

## 📸 Preview
Netflix-inspired UI with modern design and smooth interactions.

## ⚠️ Note
API key is removed for security. Add your TMDB API key in `script.js` to run locally.
